# 🛠️ PRPs-Framework Setup Guide (Symlink Approach)

คู่มือการตั้งค่า Framework โดยใช้การทำ **Symbolic Link (Symlink)** เพื่อให้สามารถใช้ Framework จากศูนย์กลางได้กับหลายโปรเจกต์

---

## 1. 📥 Git Clone (ดาวน์โหลด Framework)
เลือกโฟลเดอร์ส่วนกลางที่ต้องการเก็บ Framework ไว้ (เช่น `C:\Tools` หรือใน `d:\wsl\`)

```powershell
# ตัวอย่างการ Clone ไว้ที่ d:\Tools
cd d:\Tools
git clone https://git.nstda.or.th/application-etc/rules-development.git PRPs-Framework
```

---

## 2. 🔗 ทำ Symbolic Link เข้าสู่ Project
ย้ายไปที่ **Project ของคุณ** แล้วสร้าง Link เชื่อมโยงโฟลเดอร์หัวใจของ Framework เข้ามา

> [!IMPORTANT]
> การสร้าง Symlink บน Windows ต้องเปิด Terminal ด้วยสิทธิ์ **Administrator**

### **Windows (PowerShell):**
```powershell
# เข้าไปที่โฟลเดอร์โปรเจกต์ของคุณ
cd path\to\your-project

# สร้าง Symlink เชื่อมไปที่โฟลเดอร์ .agent (หรือ .cursor)
# New-Item -ItemType SymbolicLink -Path "{ชื่อที่จะใช้ในโปรเจกต์}" -Target "{ที่อยู่จริงของ Framework}"
New-Item -ItemType SymbolicLink -Path ".agent" -Target "d:\Tools\PRPs-Framework\.agent"
```

### **Linux / Mac / WSL:**
```bash
ln -s /path/to/PRPs-Framework/.agent .agent
```

---

## 3. 📄 คัดลอกไฟล์ตั้งต้นที่จำเป็น
เพื่อให้ AI IDE รู้จัก Framework คุณต้องมีไฟล์ควบคุมอยู่ที่ Root ของโปรเจกต์:

```powershell
# คัดลอกไฟล์หลักจาก Framework มาไว้ที่โปรเจกต์ของคุณ
cp d:\Tools\PRPs-Framework\active-ide.py .
cp d:\Tools\PRPs-Framework\INITIAL.md .
cp d:\Tools\PRPs-Framework\.antigravityrules .
```

---

## 4. 🚀 เริ่มใช้งานและซิงค์ข้อมูล (Activate)
เมื่อทำ Symlink และคัดลอกไฟล์เสร็จแล้ว ให้รันสคริปต์เพื่อให้ Framework ตั้งค่าลิงก์ภายในให้ถูกต้องตาม IDE ที่คุณเลือกใช้:

```powershell
# สำหรับ Antigravity
python active-ide.py --antigravity

# สำหรับ Cursor
python active-ide.py --cursor
```

## ✅ ตรวจสอบความพร้อม
เมื่อทำครบขั้นตอนแล้ว คุณควรจะเห็น:
1. โฟลเดอร์ `.agent` ในโปรเจกต์ (ที่มีสัญลักษณ์ลูกศร symlink)
2. ไฟล์ `INITIAL.md` และ `.antigravityrules` อยู่ที่ root
3. สามารถเรียกคำสั่ง `/00-Init` ในช่องแชทของ AI เพื่อเริ่มวิเคราะห์โปรเจกต์ได้ทันที
