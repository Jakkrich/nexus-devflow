## SERENA MCP TOOL

Use the Serena MCP server for automated validation, research, and task execution.

### Overview
Serena is a comprehensive MCP tool designed to assist with project-wide tasks, including code analysis and automated interaction.

### Usage in PRPs
When a PRP requires automated validation (e.g., in the "Validation Loop" section), use Serena tools in conjunction with local verification commands.

### Key Capabilities
- **Project Analysis**: Understanding structure and logic.
- **Task Automation**: Executing repetitive development tasks.
- **Integration**: Works seamlessly within the Cursor environment.

### Prerequisites
- Configured in `.cursor/mcp.json`.
- Project path defined in the connection arguments.

*Note: This is the primary internal MCP tool for the PRP Framework.*
