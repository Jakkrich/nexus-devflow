## 🛠️ How AI Uses Skills
The AI will automatically retrieve instructions from the `SKILL.md` files within these folders when:
1. **Context Match**: A task or question matches a specific technology stack.
2. **Workflow Start**: During core phases like `/01-Task`, `/02-Plan`, or `/03-Code`.
3. **Explicit Call**: When you command the AI to "use Skill [name]".

## 🔄 Multi-IDE Support
Skills are mobile! Use the **`active-ide.py`** script at the project root to switch the framework between:
- **Cursor** (`.cursor/skills/`)
- **Antigravity/Agent** (`.cursor/skills/`)
- **Windsurf** (`.cursor/skills/`)

## 🛡️ Pure Agentic Standards
All Skills are architected for **Zero-Script** operations:
- **Independent**: AI modifies files directly based on knowledge, not external scripts.
- **Valdating**: Enforces a **Validation Loop** to catch errors early.
- **Traceable**: All progress is logged in the project's `.auto-claude/` directory.
