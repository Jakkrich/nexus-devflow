# Utility Tools

General utility tools for insights, fixes.

## Usage

```
/utility-tools [tool]
```

## Arguments

- **tool**:
  - `insight`: Insight Extractor (`insight_extractor.md`)
  - `qa-fixer`: QA Fixer (`qa_fixer.md`)
  - `val-fixer`: Validation Fixer (`validation_fixer.md`)

## Process

1. **Load Prompt**: Load the corresponding prompt file.
2. **Execute**: Run the utility.
