# Strategic Analysis

Analyze competitors and plan long-term roadmaps.

## Usage

```
/strategic-analysis [type]
```

## Arguments

- **type**: The type of analysis.
  - `competitor`: Competitor Analysis (`competitor_analysis.md`)
  - `roadmap-features`: Feature Roadmap (`roadmap_features.md`)
  - `roadmap-discovery`: Discovery Roadmap (`roadmap_discovery.md`)

## Process

1. **Load Prompt**: Based on the `type`, load the corresponding system prompt from `PRPs-Framework/prompts/`.
2. **Execute**: Run the analysis or planning session.
