# Project Management

Tools for project assessment and planning.

## Usage

```
/project-management [tool]
```

## Arguments

- **tool**:
  - `complexity`: Complexity Assessor (`complexity_assessor.md`) - Estimate task complexity.
  - `followup`: Follow-up Planner (`followup_planner.md`) - Plan next steps.

## Process

1. **Load Prompt**: Load `complexity_assessor.md` or `followup_planner.md`.
2. **Execute**: Run the assessment or planning.
