# Advanced Spec Tools

Tools for refining and accelerating spec creation.

## Usage

```
/advanced-spec [tool]
```

## Arguments

- **tool**:
  - `critic`: Spec Critic (`spec_critic.md`) - Review and critique a spec.
  - `quick`: Quick Spec (`spec_quick.md`) - Create a spec rapidly.
  - `gatherer`: Requirement Gatherer (`spec_gatherer.md`) - Interview for requirements.

## Process

1. **Load Prompt**: Load the corresponding prompt.
2. **Execute**: Perform the specific spec task.
