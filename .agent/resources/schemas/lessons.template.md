## How To Add A Lesson #section/guide

### When To Add #section/guide

- **After receiving Feedback/Reject**: When a human corrects a styling or logic mistake made by AI.
- **After successful completion (Approve)**: When a great solution or pattern should be reused.
- **When encountering recurring bugs**: When debugging is needed to solve the same problem repeatedly.

### Who Should Add #section/guide

- **Agent**: Adds records during approval, rejection, follow-up, or insight extraction workflows.
- **Developer**: Can manually add records when specific criteria need emphasis.
